This is a simple recipe website built as the first project in The Odin Project Foundations course (HTML)
This project demonstrates basic web development skills using only HTML.

Project Overview:
-Main page ('index.html') with links to individual recipe pages
-Multiple linked HTML files in a 'recipes/' folder.
-Semantic HTML elements (headings, paragraphs, unordered/ordered lists, images, links)
-Proper file structure and relative paths

Recipes Included:
-Neer Dosa
-Chicken Ghee Roast
-Goli Bajje

-Skills Demonstrated 
By completing this project, I've practiced and shown:
-Writing valid, demantic HTML5
-Creating hyperlinks between pages
-Organizing content with headings, lists (ingredients & steps), and images
-Using proper file structure
